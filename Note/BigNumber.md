### 无符号运算关系表，以及关于0进行加速运算

| 双目运算 | abs(a) | abs(b) | ans             | example       |
| :------: | :----: | :----: | --------------- | ------------- |
|    +     |   0    |   b    | b               | 0+1=1         |
|    +     |   a    |   0    | a               | 1+0=1         |
|    -     |   a    |   0    | b               | 1-0=1         |
|    *     |   0    |   b    | 0               | 0*1=0         |
|    *     |   a    |   0    | 0               | 1*0=0         |
|    /     |   0    |   b    | 0               | 0/1=0         |
|    /     |   a    |   0    | nil             | error 1/0=nil |
|    /     |   a    |   b    | a >= b -> a / b | 3/2=1         |
|    /     |   a    |   b    | a < b-> 0       | 2/3=0         |
|    %     |   0    |   b    | 0               | 0%2=0         |
|    %     |   a    |   0    | nil             | error 2%0=nil |
|    %     |   a    |   b    | a>=b -> a%b     | 3%2=1         |
|    %     |   a    |   b    | a<b -> a        | 2%3=2         |
